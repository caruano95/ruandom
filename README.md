# ruandom

Personal site at [ruandom.com](https://www.ruandom.com). Built with plain HTML/CSS/JS, hosted on GitHub Pages.

## Structure

```
index.html          Homepage with room navigation
garage.html         The Garage — owned cars, rentals, car spotting
assets/
  css/style.css     All styles
  images/
    cars/           Owned car photos
    rentals/        Rental car photos
    spots/          Car spotting photos
```

## Adding content

Edit the relevant HTML file directly. Car descriptions are in `<p class="car-note">` tags, rental notes in `<p class="rental-note">`, spotting notes in `<p class="spot-note">`.

## Local preview

Open any `.html` file directly in a browser — no build step needed.
